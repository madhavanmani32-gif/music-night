# FUTURE // UNDER CONSTRUCTION

A futuristic college project showcasing creativity, innovation, and AI-powered digital experiences.

## Project Overview

"FUTURE // UNDER CONSTRUCTION" is a web-based project that combines AI-generated visuals, AI-assisted music creation, and modern web development to represent the journey of future engineers.

## Features

- Responsive Homepage
- AI-Generated Poster
- AI-Assisted Music Track
- Team Details
- Creative Storytelling
- Futuristic Design
- Mobile-Friendly Interface
- Live Deployment

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Gemini
- Google AI Studio
- Render
